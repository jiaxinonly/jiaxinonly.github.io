Kubernetes集群高可用二进制部署-基础篇

虚拟机软件：VMware Workstation 16 Pro
操作系统：CentOS 7.5(1804) 最小安装
Kubernetes：1.23.16

1.23以上版本将不支持docker

适用于生产环境 完全手动配置和部署 便于了解和学习kubernetes

部署规划

IP规划

IP地址 用途
192.168.66.110 vip
192.168.66.111 master01
192.168.66.112 master02
192.168.66.113 master03
192.168.66.121 node01
192.168.66.122 node02

path Default CN description

front-proxy-ca.crt,front-proxy-ca.key kubernetes-front-proxy-ca For the front-end proxy

启动kube-proxy

  5. 根据上一步获取的name查询token的值

  6. 设置客户端认证参数，token为上一步获取的值

  7. 设置上下文参数
